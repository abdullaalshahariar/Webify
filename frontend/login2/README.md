# Login Page 2 - Alternative Login Interface

An alternative visual login interface with card-based design for WEBIFY.

## Overview

This is a secondary login page that provides a more visual, card-based approach to user authentication. It features a laptop image showcase and simplified login options.

## Files

- **index.html** - Card-based login page structure
- **styles.css** - Visual styling with animations and card effects
- **script.js** - Navigation logic for login and signup flows
- **back.jpg** - Background image for visual appeal
