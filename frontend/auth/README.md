# Authentication System

## Files

- **login.html** - Combined login and signup page with tab switching
- **login.js** - Authentication logic and form validation
- **forgot-password.html** - Password recovery page
- **forgot-password.js** - Password reset functionality
- **auth.css** - Shared styles for all authentication pages
